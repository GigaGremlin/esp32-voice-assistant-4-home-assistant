# GigGremlin Firmware Personalizer GUI

**Dieses kleine GUI soll dabei helfen die Firmware einfacher an die persönlichen Bedürfnisse im Netzwerk anzupassen.**
**So kann der Name und die IP schnell geändert werden. Das ist gerade bei mehreren Geräten hilfreich und spart Zeit.**
**Darüber hinaus kann ich nur jedem dazu raten eine feste IP für seine VA’s zu verwenden.**

This small GUI is designed to help you easily adapt the firmware to your individual network needs. 
For example, you can quickly change the name and IP address. This is especially helpful with multiple devices 
and saves time. Furthermore, I strongly recommend using a static IP address for your virtual assistants.
